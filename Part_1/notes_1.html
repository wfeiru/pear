Part 1 - HTML

In this part of the workshop, you will learn what HTML is and how HTML is structured.

1.1 - HTML Notes

  • All webpages are written in HyperText Markup Language (HTML)
    • Either dynamically produced or statically written ahead of time
  • HTML consists of content enclosed in nested, predefined tags
    • Tags define the structure and role of content
    • Ex: <p>Hi there</p> tells the browser that “Hi there” is a paragraph
  • No one is expected to know every tag
    • Learn a handful of the most common tags, lookup others as you go
    • www.w3schools.com/tags

1.2 - Basic HTML File Structure

  • <!DOCTYPE html>: Tells browser we are using HTML5; a void, empty tag (no closing tag, no content)
  • <html>: Starts HTML content
    • <head>: Defines “web page settings”
      • <title>: Page title, as seen on browser tab
      • <link>: Links to other, non-HTML resources we may need in this page
    • <body>: Starts page content
      • <nav>: General container for any “navigation content”
      • <section>: General container for a “section” of content
      • <footer>: General container for “footer” content

1.3 - Adding to Code

  • Your goal is to make this page look like this page by adding the necessary HTML elements.
  • Some elements you might need include...
    • <div> - Generic container
      • <main>, <section>, <footer>, etc. are variants of <div>
    • <a href=””> - Hyperlinks
    • <p> - Paragraphs
    • <img src=””> - Image
    • <ol> - Numbered list
    • <li> - Item in a list
    • <h1>, <h2>, … - Headings
    • <button> - Generic button
