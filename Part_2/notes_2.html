Part 2 - CSS

In this part of the workshop, you will learn what CSS is and how to apply your own CSS styles.

2.1 - CSS Basics

  • HTML defines what content is on a page, Cascading Style Sheets (CSS) define how it appears
  • General syntax:
    selector {
       attributeName: attributeValue;
    }
  • Ex:
    body {
       background-color: pink;
    }
  • Selector is often a tag name, such as body or button
  • Styles are applied in the order in which they are assigned and by how specific they are; read more at www.w3schools.com/css

2.2 - Classes, External Stylesheets

  • If we want to apply styles more generally, we can assign class names to tags, and use the class as a selector instead
    • Ex:
      <p class="center">A centered paragraph</p>
      <h1 class="center">A Centered Heading</h1>
      …
      .center {
         text-align: center;
      }
  • Best practice is to put styles in a .css file and link to it using <link> inside the <head> of your webpage
    • Ex:
      <head>
         …
         <link rel="stylesheet" href="styles.css">
      </head>

2.3 - Adding to Code

  • Your goal is to make this page look like this page by adding the necessary CSS.
  • Some CSS attributes you might need include...
    • background-color
    • color
    • font-size
    • font-family
    • text-align
    • width
