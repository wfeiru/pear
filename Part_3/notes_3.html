Part 3 - Bulma

In this part of the workshop, you will learn what a CSS Framework is and how to apply styles using Bulma.

3.1 - CSS Frameworks

  • 3rd-party CSS frameworks let you use predefined styles others have published
  • Pearl Hacks website uses Bootstrap; we’ll be using Bulma
  • General Steps:
    • Link the framework using <link>
      • The framework’s documentation will give you guidance and also show you how to link other necessary dependencies
    • Style your HTML using class names provided by the framework
      • Again, reading documentation will help you find these classes

3.2 - Getting Started With Bulma

  • Go to https://bulma.io/documentation/overview/start/
  • Use the CDN link to grab Bulma
    • In the <head> tag, add the following:
      <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.8.0/css/bulma.min.css">
  • Bulma also recommends adding the following tags to <head>as well:
    • <meta name="viewport" content="width=device-width, initial-scale=1">
    • <script defer src="https://use.fontawesome.com/releases/v5.3.1/js/all.js"></script>

3.3 - Adding to Code

  • Your goal is to make this page look like this page by adding the necessary Bulma classes.
  • Some Bulma classes you might need include...
    • .title, .subtitle
    • .is-danger
    • .columns, .column
    • .nav, .footer
    • .card
